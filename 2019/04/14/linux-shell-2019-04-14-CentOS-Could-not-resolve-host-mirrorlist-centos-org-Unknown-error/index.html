Could not resolve host：mirrorlist.centos.org; Unknown error

发表于 2019-04-14 • 分类于 问题记录

执行 yum 命令时报：

配置下网络即可。

  • 查看网卡配置文件：

    第一个 ifcfg-ens32 就是网卡配置文件，不同机器略有不同：

  • 备份

  • 编辑 ifcfg-ens32 文件：

    • 针对动态配置，需要设置：

      BOOTPROTO=static
      ONBOOT=yes

      IPADDR=192.168.1.74
      GATEWAY=192.168.1.1
      NETMASK=255.255.255.0
      DNS1=8.8.8.8

      $ sudo systemctl restart network
      $ sudo yum update
