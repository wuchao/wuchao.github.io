MySQL 数据类型转换

MySQL 也提供了两个显示转换的函数：CAST(value as type) 和 CONVERT(value, type)。

value：为要转换原数据类型的值。

type：目的类型。但可以转换的类型是有限制的。类型可以是以下任何一个：

  • 二进制，同带 binary 前缀的效果：BINARY
  • 字符型，可带参数：CHAR()
  • 日期：DATE
  • 时间：TIME
  • 日期时间型：DATETIME
  • 浮点数：DECIMAL
  • 整数：SIGNED
  • 无符号整数：UNSIGNED
